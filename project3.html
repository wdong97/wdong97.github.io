generative visuals for live sets

i produce electronic music, and i wanted visuals that react to what i'm actually playing instead of looping the same clip behind a set. this is a small engine that turns live audio into generative visuals.

it takes an audio input, pulls beat and frequency information out of it in real time, and uses that to drive shader-based visuals that move with the track. i built it to run in the browser so i can bring it to a set with just a laptop.

next steps: more presets, and midi control so i can switch scenes from a controller mid-set.
